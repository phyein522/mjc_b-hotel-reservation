2시즌 기본 정보
● 예정 (D-3)
⚠️ 입력한 기간이 기존 ‘준성수기(07.01~07.16)’와 겹치지 않아요. 겹치는 경우 저장 시 자동으로 알려드려요.
상태는 관리자가 고르지 않고 날짜 기준으로 시스템이 자동 계산해요. (오늘보다 미래=예정, 기간 안=진행중, 지남=종료)
3요금 설정
요일별 요금 정책 적용
켜면 '요일별 요금 정책'의 금/토/일 할증(%)이 이 시즌 기준요금에 자동으로 곱해져요
껐을 경우에만 위의 '주말요금 직접입력'이 활성화되고, 그 값이 그대로 주말 가격이 돼요. 두 방식은 동시에 쓸 수 없어요 (계산 충돌 방지).
